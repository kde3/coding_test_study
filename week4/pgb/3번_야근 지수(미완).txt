-> a+b+c = k일때, 각 제곱의 합이 최소가 되려면 a=b=c를 향해 가는 그리디 알고리즘

def solution(n, works):

    left_work = sum(works)-n
    
    if(len(works) == 1):
        return (max(works[0]-n,0)**2)
    
    if sum(works) < n:
        return 0
    
    for _ in range(n):
        m = works.index(max(works))
        works[m] -= 1
        
    answer = sum([x**2 for x in works])
    return answer

-> n번동안 최대값을 찾아서 1씩 빼기
